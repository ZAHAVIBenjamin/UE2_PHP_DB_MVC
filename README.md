# 🏰 Les Chasses de Lorthan le Fabuleux

Une application web immersive de type "Chasse au Trésor" réalisée dans le cadre de ma formation. Ce projet simule une plateforme d'aventure où les utilisateurs peuvent s'inscrire, acheter des parchemins magiques via une boutique, et débloquer des quêtes spécifiques.

## 🌟 Fonctionnalités clés

* **Système d'Authentification :** Gestion complète de l'inscription et de la connexion des utilisateurs via `localStorage`.
* **Boutique & Panier :** * Vitrine de produits (parchemins et goodies).
    * Gestion dynamique du panier (ajout, suppression, calcul du total).
    * Persistance des achats dans l'inventaire du profil utilisateur.
* **Intégration d'API Externe :** Consommation de l'API Harry Potter pour afficher et naviguer parmi les personnages de la saga.
* **Animations Interactives :** Effets de survol (hover) personnalisés avec des éléments graphiques (système de clef animée dans le menu).
* **Interface Responsive :** Design adapté aux différentes tailles d'écran (Desktop, Tablette, Mobile) via des Media Queries CSS.
* **Espace Contact :** Formulaire dynamique accessible uniquement aux utilisateurs connectés.

## 🛠️ Technologies utilisées

* **HTML5 / CSS3 :** Structure et mise en forme avec l'utilisation de polices Google Fonts (*Cinzel Decorative*).
* **JavaScript (ES6+) :** Logique métier, manipulation du DOM, gestion de l'asynchrone (Fetch API).
* **Stockage :** `localStorage` pour la simulation d'une base de données côté client.
